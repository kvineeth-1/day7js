An array is a collection of items stored at contiguous memory locations.

                const a=[1,'Hi',3,7]
                console.log(a);


                Map
                It iterates through each elements of the array and do the required operations.
                var arr=[1,2,3,4,5]
                var result=arr.map((x)=>x*2)
                console.log(result);
               

                Filter
                Applies condition and returns a new array
                const r=arr.filter((x)=>x>3)
                console.log(r);


                Reduce
                returns a single value
                var result=arr.reduce((x,y)=>x=x+y,0)
                console.log(result)

                Push
                Add Element in Last.
               
                Pop
                Remove element from last.

                Unshift
                Add element in starting.

                Shift
                Remove first element.

                some:if one condition is true then it will return true.
                every:All the condition should be satisfied then only it will return true.
